# Proofreading

## Instructions

- Correct spelling mistakes.
- Correct punctuation errors.
- Correct grammar mistakes.
- Preserve substance, meaning, style, tone, structure, and wording.
- Do not rewrite, rephrase, summarize, expand, or improve beyond necessary corrections.
- If a correction is ambiguous, ask the user before changing it.
